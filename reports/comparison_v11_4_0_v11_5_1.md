# Comparison Report: Mattermost v11.4.0 vs v11.5.1

This report summarizes the changes, improvements, and security hardening between Mattermost versions v11.4.0 and v11.5.1, based on an analysis of 178 commits and over 1,500 modified files.

## 1. Security Hardening
Version 11.5.1 specifically focused on critical and medium-level security fixes:
- **Authentication Update Protection:** Hardened the `updateUserAuth` API endpoint to use standard session handlers, preventing potential account takeover vectors.
- **Guest Magic Links:** Implemented atomic token consumption for guest magic links to prevent replay attacks.
- **SSRF Mitigation:** Fixed an SSRF bypass involving IPv4-mapped IPv6 literals.
- **Rate Limiting:** Added rate limiting to the login endpoint (5 requests/second, 10 burst) to mitigate brute-force attempts.

## 2. AI & Productivity Features
Significant advancements were made in the AI integration layer:
- **Context-Aware AI Rewrites:** The "AI Rewrites" feature now includes context from the last 10 messages in a thread to provide more accurate and relevant responses.
- **Auto-translation (Beta):** Introduced support for automatic message translation for Enterprise Advanced users, with improvements to worker efficiency (up to 64 workers).
- **AI Recaps:** Released the MVP for channel and thread summarization (Recaps) with dedicated UI icons and citation support.

## 3. Technical Infrastructure
- **Node.js 24 Upgrade:** The webapp infrastructure was upgraded to Node 24, alongside updates to Webpack 5 and Jest 30.
- **PostgreSQL CJK Support:** Native support for Chinese, Japanese, and Korean (CJK) search in PostgreSQL databases.
- **Property Access Service:** Integrated a new service to manage read/write access to user property fields, allowing for "protected" fields managed by specific plugins.

## 4. User Experience (UX/UI)
- **RHS Enhancements:** Users can now rename channels and access channel settings directly from the Right-Hand Sidebar (RHS).
- **File Handling Changes:** PSD (Photoshop) files are no longer previewed inline to prevent OOM (Out of Memory) issues with large files.
- **New Icons:** Distinct icons for archived and private channels were added for better visual differentiation.
- **Tooltips:** Action buttons now feature descriptive tooltips and display error messages more clearly.

## 5. Key Bug Fixes
- **Import Process:** Fixed an issue where the import process failed to find main JSON files if they were in subfolders.
- **WebSocket Stability:** Fixed an issue where thread read states were not correctly updated over WebSockets.
- **Scrolling Behavior:** Resolved a bug where the post list would jump to the bottom when editing older messages.
